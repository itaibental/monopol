# 🎲 בגרות מונופול — תקשורת וחברה תשפ"ו

משחק לוח אינטראקטיבי ללימוד 33 מושגי מפתח בתקשורת וחברה לקראת בחינת הבגרות.
מחובר לשרת Firebase עם שמירת התקדמות, טבלת מובילים וממשק ניהול למורה.

## מבנה הפרויקט

```
├── index.html          # דף פתיחה — כניסה למשחק / לניהול
├── game.html           # המשחק עצמו עם לוגין קבוצתי
├── admin.html          # ממשק ניהול למורה (עריכת שאלות + דירוג)
├── styles.css          # עיצוב הלוח (CSS)
├── data.js             # נתוני מושגים, אזורים וחידונים (ברירת מחדל)
├── board.js            # לוגיקת הלוח, פריסה ומיקום תאים
├── firebase-config.js  # הגדרת Firebase + פונקציות DB
├── .nojekyll           # תמיכה ב-GitHub Pages
└── README.md
```

## תכונות

- **מסך פתיחה** — כניסה למשחק או לאזור ניהול
- **הזדהות קבוצתית** — שחקנים בוחרים שם קבוצה וחוזרים איתו
- **משתתף בודד / קבוצתי** — שני מצבי משחק
- **שמירת התקדמות** — Firebase Realtime Database
- **טבלת מובילים** — מעודכנת בזמן אמת
- **ממשק ניהול** — עריכת שאלות, הוספת שאלות, צפייה בדירוג
- **סיסמת מורה** — ברירת מחדל: `teacher2026`

## הפעלה

1. העלה את כל הקבצים ל-repository ב-GitHub
2. Settings → Pages → Source: `main`, folder: `/ (root)`
3. האתר זמין ב: `https://<username>.github.io/<repo-name>/`

## Firebase

הפרויקט משתמש ב-Firebase Realtime Database.
הגדרות החיבור נמצאות ב-`firebase-config.js`.
